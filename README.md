# kevindelord.io

## Local environment

To reboot on a local machine:

```
$> ./reboot.sh
```

And test using:

http://127.0.0.1:4000/

## Add a new post

Execute the following script:

```
$> ./create_new_post.py
```

It will create a new `.md` file ready to be filled!
